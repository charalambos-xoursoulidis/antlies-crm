Αποθήκευση στον Server ΣΥΝΙΣΤΑΤΑΙ
Τα δεδομένα αποθηκεύονται ως αρχεία JSON στον server (crm_customers.json, crm_quotes.json). Κοινή πρόσβαση από όλους τους χρήστες. Απαιτεί PHP.
☁️
Supabase Cloud
Δωρεάν cloud βάση δεδομένων. Για πρόσβαση από διαφορετικές τοποθεσίες/δίκτυα.
💻
Τοπική αποθήκευση (browser)
Αποθήκευση στον browser. Μόνο για δοκιμές — τα δεδομένα δεν μοιράζονται.

Δημιουργήστε δωρεάν project στο supabase.com και τρέξτε αυτό το SQL:

CREATE TABLE customers ( id UUID DEFAULT gen_random_uuid() PRIMARY KEY, created_at TIMESTAMPTZ DEFAULT NOW(), lastname TEXT NOT NULL, firstname TEXT, address TEXT, city TEXT, zip TEXT, phone TEXT, email TEXT, notes TEXT ); CREATE TABLE quotes ( id UUID DEFAULT gen_random_uuid() PRIMARY KEY, created_at TIMESTAMPTZ DEFAULT NOW(), customer_id UUID REFERENCES customers(id) ON DELETE CASCADE, quote_number TEXT, date DATE, status TEXT DEFAULT 'draft', notes TEXT, brand TEXT, brand_label TEXT, type_idx INT DEFAULT 0, pump_id TEXT, pump_name TEXT, pump_kw TEXT, sections JSONB, totals JSONB ); ALTER TABLE customers ENABLE ROW LEVEL SECURITY; ALTER TABLE quotes ENABLE ROW LEVEL SECURITY; CREATE POLICY "allow_all" ON customers FOR ALL USING (true) WITH CHECK (true); CREATE POLICY "allow_all" ON quotes FOR ALL USING (true) WITH CHECK (true);
